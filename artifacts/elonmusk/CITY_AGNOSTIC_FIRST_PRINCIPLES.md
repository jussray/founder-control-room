# Elon Musk — First-Principles City-Agnostic Artifact

## Fundamental outcome

A jurisdiction should be able to turn scattered evidence into one ranked, auditable action queue without buying a city-specific software rewrite.

## Requirement challenge

- Delete the assumption that the first customer defines the schema.
- Delete separate scoring logic by city.
- Delete automatic fallback to a default jurisdiction.
- Delete direct browser access to privileged economic operations tables.
- Delete unsourced claims and unversioned scores.
- Delete automation that submits, funds, or approves before the human authority loop is proven.

## Simplified system

```text
jurisdiction + sources + programs + signals
  -> deterministic score
  -> founder/local-authority decision
  -> pilot
  -> measured outcome
```

## Constraints that remain real

- local law and program eligibility;
- evidence freshness and completeness;
- staff capacity;
- provider credentials and deployment authority;
- privacy and public-record boundaries;
- infrastructure cost and reliability.

## Fastest truthful proof

Use the same route and score function for Johnstown and a visibly synthetic second jurisdiction. Identical signals must produce identical scores. Unknown jurisdictions must fail closed. Playwright must attack leakage and malformed input.

## Automate last

Automate source refresh and evidence-staleness alerts before automating applications, funding decisions, public publication, or external follow-up.
